# Instagram DM Toplu Silici Chrome Eklentisi

Bu Chrome eklentisi, Instagram web arayüzünde DM (Direct Message) kutunuzdaki sohbetleri toplu olarak silmenizi sağlar.

## Kurulum

1. Bu projeyi bilgisayarınıza indirin
2. Chrome tarayıcınızda `chrome://extensions/` adresine gidin
3. Sağ üst köşedeki "Geliştirici modu"nu açın
4. "Paketlenmemiş öğe yükle" butonuna tıklayın
5. İndirdiğiniz proje klasörünü seçin

## Kullanım

1. Instagram web sitesinde DM (mesajlar) sayfasını açın
2. DM kutusunun üst kısmında "Tüm DM'leri Sil" butonunu göreceksiniz
3. Butona tıklayarak tüm sohbetleri topluca silebilirsiniz

## Uyarı

- Bu işlem geri alınamaz! Silinen sohbetler kurtarılamaz.
- Hesabınızın güvenliği için işlemler arası kısa bekleme süresi bırakılmıştır.
- Instagram arayüzü değişirse eklenti çalışmayabilir.

## Not

Bu eklenti sadece kendi hesabınızda çalışır ve başka kullanıcıların DM'lerini silemez.
